Local Setup:

* Prerequisites for running locally
  * node.js
  * ruby

* Command line install dependencies
  * gem install compass
  * npm install
  * bower update

* Running the app
  * grunt serve

* Building for distrubution
  * grunt build

* Unit tests
  * grunt test

* Functional tests
  * grunt protractor
  
Configuration:
* config.js - root - change the voyager instance
